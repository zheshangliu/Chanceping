# 贡献者行为准则

## 我们的承诺

为了营造一个开放友好的社区环境，我们作为贡献者和维护者承诺：参与本项目的每个人，无论年龄、体型、残疾状况、种族、性别认同与表达、经验水平、教育程度、社会经济地位、国籍、个人外貌、种族、宗教或性取向，都将免受骚扰。

我们承诺以促进开放、包容、多元化、健康社区的方式行动和互动。

## 我们的准则

有助于为本社区营造积极环境的行为示例：

- 对他人展示同理心和善意
- 尊重不同意见、观点和经验
- 给予建设性反馈，也优雅地接受反馈
- 承认错误、道歉，并从中学习
- 关注的不仅是个人，更是整个社区的福祉

不可接受的行为示例：

- 使用性化语言或意象，以及任何形式的性关注或骚扰
- 钓鱼、辱骂或贬损的评论；人身或政治攻击
- 公开或私下的骚扰
- 未经明确许可发布他人私人信息（如物理地址或电子邮箱）
- 其他在专业环境下可被合理认定为不当的行为

## 执行责任

社区领导者负责澄清和执行本行为准则的标准，并将对其认为不当、威胁、冒犯或有害的任何行为采取适当且公平的纠正措施。

## 适用范围

本行为准则适用于所有社区空间，并在个人在公共空间正式代表社区时同样适用。

## 执行

如遇滥用、骚扰或其他不可接受的行为，可通过以下方式举报：

- 提交私有 issue（标记为 `conduct`）
- 发送邮件至项目维护者（见仓库 profile）

所有举报都会被及时且严肃地审查和处理。社区领导者有义务为举报者保密。
